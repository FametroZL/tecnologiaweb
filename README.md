# Repositório: TecnologiaWeb

Bem-vindos! Este é o repositório oficial da disciplina de **Tecnologia Web**, ministrada pelo Prof. Alexsander Souza.

Aqui, vocês deverão enviar os seus portfólios desenvolvidos com **HTML** e **CSS**.

---

## 📌 Instruções para os Alunos

Siga o passo a passo abaixo para enviar o seu portfólio (`index.html` e `style.css`) para este repositório:

### 1. Aceite o convite no seu e-mail
Você recebeu um convite no seu e-mail para participar da organização no GitHub. Clique no link do e-mail para **Aceitar o convite** (Você precisará ter uma conta no GitHub e estar logado nela).

### 2. Preparando os seus arquivos
Antes de enviar, certifique-se de que os seus arquivos estão organizados no seu computador.
- Garanta que você possui os arquivos **`index.html`** e o seu arquivo de estilização **`style.css`**.
- Se você utilizou imagens externas, prepare também a pasta `img` (ou `images`) contendo as imagens do seu projeto.

### 3. Enviando o seu Portfólio (Upload Direto)

Como você já faz parte da organização, a dinâmica de envio simplificou! Siga os passos:

#### Método A: Pelo Navegador (Mais fácil)

1. Acesse este repositório: [https://github.com/FametroZL/TecnologiaWeb](https://github.com/FametroZL/TecnologiaWeb)
2. Clique no botão **"Add file"** (Adicionar arquivo) e depois em **"Create new file"** (Criar novo arquivo), para que você consiga criar a sua pasta.
3. No campo de nome do arquivo ("Name your file..."), digite o seu nome e sobrenome, tudo minúsculo e separado por traço, seguido por uma barra de espaço. (Exemplo: digite `joao-silva/`). 
   *Note que ao digitar a barra `/`, o GitHub automaticamente criará a sua pasta.*
4. Continue no campo de texto e crie o seu `index.html`. Copie e cole o código do seu projeto no quadro de texto que abrirá.
5. Role até o final da página e clique no botão verde **"Commit changes"**.
6. Para adicionar os outros arquivos (`style.css` e as pastas de imagens), basta abrir a pasta com o seu nome (`joao-silva`) que acabou de ser criada, clicar em **"Add file"** > **"Upload files"**.
7. Arraste os outros arquivos (`style.css`, etc.) para a tela.
8. Role a tela para baixo e clique no botão verde **"Commit changes"**.

🎉 Pronto! Os seus arquivos estão na sua pasta dentro do repositório da turma!

---

#### Método B: Usando Git no Terminal (Desafio de Mercado)

Quer se desafiar e enviar o seu código pelo terminal (Git Bash)? Siga este caminho:

1. Abra o terminal no seu computador e faça o clone deste repositório oficial:
   ```bash
   git clone https://github.com/FametroZL/TecnologiaWeb.git
   ```
2. Abra a pasta baixada no seu editor de código (VS Code, etc). Crie dentro dela a sua pasta pessoal com seu nome e sobrenome (ex: `joao-silva`).
3. Coloque os seus arquivos do projeto (`index.html`, `style.css`, etc.) dentro desta sua nova pasta.
4. Adicione as mudanças, crie o commit e envie de volta para o GitHub:
   ```bash
   git add .
   git commit -m "feat: adiciona portfólio de João Silva"
   git push origin main
   ```

🎉 Pronto! Seu envio foi computado direto pelo terminal.
